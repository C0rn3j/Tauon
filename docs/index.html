Tauon Music Box    

A streamlined music player for the desktop

v2.5.1
View Github page

Features

  • Easy drag and drop track importing
  • Supports MP3, FLAC and other codecs
  • Automatically detects CUE sheets
  • No hidden changes to your music library in the background
  • Visually compare individual track play times
  • Rename and delete folders straight from the UI
  • Open tracks in your favourite tag editor
  • Broadcast asynchronously like a DJ How?
  • Simple batch transcoding of albums to single folders How?
